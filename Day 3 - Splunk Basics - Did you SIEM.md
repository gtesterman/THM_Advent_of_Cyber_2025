<h3>Splunk Basics - Did you SIEM? (Day 3)</h3>

Day 3 explored the use of Splunk Enterprise to interpret log data and detect anomolies. The room provides an interactive Splunk Enterprise instance where participants can learn the funamentals of Spunk, from basic searches to visualizing the logs timeline and tracing the attack chain of abnormal web traffic. Participants used Splunk to trace an attacker IP and find vulnerabilities such as SQL injection attack and ransomware attack.

<h4>Concepts Explored:</h4>

- Interpreting Custom Log Data
- Visualizing Splunk Logs
- Search Processing Language (SPL)
- Anomaly Detection

<h4>Tools Used:</h4>

- Splunk Enterprise
