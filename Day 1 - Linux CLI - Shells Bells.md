<h1>Linux CLI - Shells Bells (Day 1)</h1>
Day 1 explores the Linux Command-line Interface (CLI) from a beginner-level standpoint. TryHackMe provides a short view course explaining the basics of Linux terminal and how it can be more used more productively than a graphical interphase (GUI). Participants will be exposed to a browser-based Ubuntu virtual machine and basic Linux commands that will help guide them through later advent rooms.

<h2>Commands Used:</h2>

- echo
- ls
- cat
- pwd
- cd
- grep
- find
- rm
- mv
- uptime
- ip addr
- ps aux
- sudo su
- whoami
- history

<h2>Tools Used:</h2>

- Ubuntu virtual machine
- Linux command-line terminal
