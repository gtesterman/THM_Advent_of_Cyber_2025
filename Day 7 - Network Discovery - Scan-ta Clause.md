<h3>Network Discovery - Scan-ta Clause (Day 7)</h3>

Day 7 explores exposed network services through Nmap. The room explains how to perform basic Nmap scans through the Linux terminal. Participants will use Nmap to discover exploitable services such as ftp, http and MySQL. They will then use the tools available to them through the Linux terminal and browser to uncover a secret message.

<h4>Concepts Explored:</h4>

- Network Discovery
- Network Protocols

<h4>Tools Used:</h4>

- Ubuntu Virtual Machine
- Nmap
- Firefox Web Browser
- Netcat
- MySQL
